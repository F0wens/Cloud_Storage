Detailed Description

The App class is the main class of the Cloud Storage application.

It extends the JavaFX Application class and defines the start() method, which is called when the application is launched. The start() method loads the login user interface defined in login.fxml and displays it in a new Stage.

The start() method is called when the JavaFX application is launched.

It loads the login.fxml file using FXMLLoader and displays the user interface defined in the file in a new Scene, which is then displayed in a new Stage.
