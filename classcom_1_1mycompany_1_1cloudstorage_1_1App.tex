\doxysubsection{Detailed Description}
The \mbox{\hyperlink{classcom_1_1mycompany_1_1cloudstorage_1_1App}{App}} class is the main class of the Cloud Storage application. 

It extends the Java\+FX Application class and defines the \mbox{\hyperlink{classcom_1_1mycompany_1_1cloudstorage_1_1App_a637f6a8eeaf87ec5dbaf06f3488c8424}{start()}} method, which is called when the application is launched. The \mbox{\hyperlink{classcom_1_1mycompany_1_1cloudstorage_1_1App_a637f6a8eeaf87ec5dbaf06f3488c8424}{start()}} method loads the login user interface defined in login.\+fxml and displays it in a new Stage.
